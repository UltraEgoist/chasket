Drop-in Components for Any Website

These Chasket components work in plain HTML. No build step, no framework migration. Just add a script tag and use the custom element.

How Drop-in Works

1

Add the Script

Include the compiled Chasket component JS file with a single script tag.

2

Use the Tag

Write the custom element tag anywhere in your HTML. It just works.

3

No Migration

Existing CSS, JS, and frameworks are untouched. Chasket is isolated in Shadow DOM.

COMPONENT

Pricing Table

A responsive 3-tier pricing table with selection state. Click a plan to see the active state animation.

HTML your-page.html 
<!-- Step 1: Add the script -->
<script src="chasket-demos.js"></script>

<!-- Step 2: Use the tag -->
<fl-pricing-table></fl-pricing-table>
COMPONENT

Contact Form

Form with client-side validation and a success state. Try submitting with empty fields to see validation.

HTML contact.html 
<!-- Works in WordPress, Squarespace, anywhere -->
<script src="chasket-demos.js"></script>

<fl-contact-form></fl-contact-form>

<!-- Your existing page content below
     is completely untouched -->
